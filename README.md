# 🛠️ update_tool

`update_tool` 是一个用于下载并解压指定压缩包到上级目录的更新工具。它旨在简化应用程序的自动更新流程。

## 📌 简介

该程序会从指定的 URL 下载压缩包，并将其解压到程序所在目录的**上一级目录**中。因此，默认情况下，`update_tool` 假设你想要更新的程序位于其上一级文件夹中。

> ⚠️ **重要提示：**
>
> 为了确保程序正常运行，请将 `update_tool` 放置在你要更新的目标程序下的 `update` 子目录中。
>
> 正确结构示例：
> ```
> your_app/
> └── update/(在此目录下运行)
> ```
